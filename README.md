# Graphpad Prism workshop, March 2026

This is principally for distribution of datasets for use in a workshop
on Graphpad Prism, run at the University of Melbourne on 23/03/2026.

All datasets have been previously published. Some of these are results
were kindly provided by A/Prof Adrian Achuthan (UniMelb), and appear
in:

Lupancu, et al. 2023. Epigenetic and transcriptional regulation of
CCL17 production by glucocorticoids in arthritis. _iScience_, **26**,
10. DOI: 10.1016/j.isci.2023.108079
